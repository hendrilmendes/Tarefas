### Recursos:

* App feito em Flutter disponível para iOS e Android
* Firebase para monitorar o desempenho, erros do app e banco de dados
* Excluir tarefas
* Fazer anotações
* Excluir anotações
* Marcar como concluído
* Edição de Tarefas e Anotações
* Agendamento de Tarefas
